Seletores CSS

Seletores CSS é a forma utilizada para aplicar estilos a tags ou elementos de uma pagina HTML. Existem três tipos de seltores, são eles:

  1. Seletor de ELEMENTOS, utilizado para estilizar todas as tags da página que sejam de um certo tipo.
  2. Seletor de id: esse seletor estiliza apenas elementos da página que possuem o id definido ; no CSS usa-se # + id.
  3. Seletor de Classe: estiliza vários elementos que sejam da mesma classe. Usa-se .nome da classe

Aplicando CSS

Existem três formas possíveis de aplicar CSS a sua página, são essas:

  • Arquivo.CSS: cria-se um link no header da página para uma folha de estilo (um arquivo de extensão CSS);
  • Aplicando diretamente na tag: utilizando a propriedade style;
  • Tag style: é uma tag onde pode ser escrito todo o estilo CSS
